Hack the Farm!

On this site you will find open source projects meant to assist with farming or life on the farm. Some of these projects will be made into a commercial products so watch out to be able to get them early! See below for the projected roadmap/timeline.

July 2023

The first version of the owl hooting project will be finished. It is also planned to make 3 simple breakout boards of potentially useful sensors.

August 2023

The owl hooting project is planned to be finished (before harvest). Work will start on the standalone LTE-M temperature monitor.

October 2023

The first version of the LTE-M standalone monitor will be done.

November 2023

The Open Grain Monitor will start to be created by using the standalone LTE-M monitor as a starting point.

January 2024

Will start looking for funding for the first production run of Open Grain Monitor.

March 2024

Start development of a LoRa gateway that also acts as a server so that no subscription is needed to monitor temperature. Again a simple standalone puck like monitor will be developed first.
